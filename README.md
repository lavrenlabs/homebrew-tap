# homebrew-tap

Homebrew formulae for [Mnemo](https://github.com/masterbor/mnemo) — persistent memory for AI coding agents.

## Install

```bash
brew install masterbor/tap/mnemo
```

## Update

```bash
brew upgrade mnemo
```

## About

This tap is auto-updated by [GoReleaser](https://goreleaser.com) on every Mnemo release.
The formula in `Formula/mnemo.rb` is generated automatically — do not edit it by hand.
